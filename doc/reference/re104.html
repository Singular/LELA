Name

uint32 — This is a representation of 32 bit unsigned ints, usually equivalent to `unsigned int'.

Synopsis

// In header: </home/hovinen/build/lela/lela/integer.h>


typedef unsigned __LELA_INT32 uint32;

Description

The use of `uint32' ensures you are working with 32 bit unsigned ints, [0..2^32). Similarly, uint8, uint16, and uint64 are defined.
