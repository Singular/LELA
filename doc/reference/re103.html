Name

int32 — This is a representation of 32 bit ints, usually equivalent to `int'.

Synopsis

// In header: </home/hovinen/build/lela/lela/integer.h>


typedef signed __LELA_INT32 int32;

Description

The use of `int32' ensures you are working with 32 bit signed ints, [-2^31..2^31). Similarly, int8, int16, and int64 are defined.
