Name

LELA::BLAS2::trsv — Triangular matrix-vector solve, x <- A^-1 x, where A is triangular.

Synopsis

template<typename Ring, typename Modules, typename Matrix, typename Vector> 
  Vector & trsv(Context< Ring, Modules > & ctx, const Matrix & A, Vector & x, 
                TriangularMatrixType type, bool diagIsOne);

Description

A must be square. If diagIsOne is false and there are zeros on the diagonal, an exception is thrown.

Parameters:

A

Matrix A

diagIsOne

Whether to assume that the entries on the diagonal of A are one

type

Whether A is upper or lower triangular

x

Vector x
