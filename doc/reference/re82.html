Description

The algorithm used is based loosely on the algorithm described in J.C. Faugère's paper "Parallel Gaussian Elimination for Gröbner bases computations in finite fields", PASCO 2010.

    Parameters:

    Matrix into which to store the reduced row-echelon form. Should have the same dimensions as A. May be the same matrix as A, in which case A is replaced by its reduced row-echelon form.

    Ring-element-reference into which to store computed determinant of pivot-submatrix
