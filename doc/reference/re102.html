Name

integer — This is a representation of arbitrary integers.

Synopsis

// In header: </home/hovinen/build/lela/lela/integer.h>


typedef mpz_class integer;

Description

It is a wrapper of GMP integers. Arithmetic operations are via C++ infix operator forms (eg. a*b). It is for ``casual'' uses such as characteristics and cardinalities and when initializing field elements. The integers are also represented as a LELA ring for use in integer matrix computation, see pid-integers.h or ntl-ZZ.h.
