Description

GMPRationalElement public construct/copy/destruct

  1. GMPRationalElement(void);
    Default constructor.

    This constructor is required to allow ring elements to be primitive C++ types. Because constructor does not know what ring the element belongs to, it cannot actually construct the element. In this implementation, the constructor it sets _elem_ptr to the null pointer. Initialization of the element is done through the ring function init where the ring is known. 

  2. GMPRationalElement(const GMPRationalElement & a);
    Copy constructor.

    This constructor is required to allow ring elements to be primitive C++ types, and to allow ring elements to be passed by value into functions. Constructs ring element by copying the ring element. In this implementation, this means copying the element to which a._elem_ptr points.

    Parameters:

    a

    ring element. 

  3. GMPRationalElement(mpq_t _rep);
    Constructor.

    Constructs ring element from an mpq_t Not part of the interface. Creates new copy of element object in dynamic memory. 

  4. GMPRationalElement(const integer & num, const integer & den);
    Constructor Initialize from numerator and denominator. 
  5. GMPRationalElement(const integer & num);
    Constructor Initalizes from a single integer, (which is assumed to be the numerator, with the denominator being 1). 
  6. GMPRationalElement& operator=(const GMPRationalElement & a);
    Assignment operator.

    Assigns element a to element. In this implementation, this is done by copying ring element to which _elem_ptr points.

    Parameters:

    a

    ring element. 

  7. ~GMPRationalElement();
    Destructor.

    In this implementation, this destroys element by deleting ring element to which _elem_ptr points.

GMPRationalElement public public data members

  1. mutable mpq_t rep;

    Not part of the common object interface for LELA} ring elements. Included to avoid code bloat.
