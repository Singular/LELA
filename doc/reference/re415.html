Name

LELA::DefaultVectorTraits — Vector traits template structure.

Synopsis

// In header: </home/hovinen/build/lela/lela/vector/traits.h>

template<typename Vector> 
struct DefaultVectorTraits {
  // types
  typedef Vector::RepresentationType RepresentationType;
  typedef Vector::StorageType StorageType;
  typedef Vector::ContainerType ContainerType;
  typedef Vector::SubvectorType SubvectorType;
  typedef Vector::ConstSubvectorType ConstSubvectorType;
  typedef Vector::AlignedSubvectorType AlignedSubvectorType;
  typedef Vector::ConstAlignedSubvectorType ConstAlignedSubvectorType;

  // public data members
  static const int align;  // Alignment factor. 
};

Description

This structure defines information about a vector on which methods may specialise and from which they may generically derive information.

DefaultVectorTraits public types

  1. typedef Vector::RepresentationType RepresentationType;

    See VectorRepresentationTypes for definitions. On this typedef methods which vary on the representation-type should specialise.

  2. typedef Vector::StorageType StorageType;

    See VectorStorageTypes for definitions.

  3. typedef Vector::ContainerType ContainerType;

    This defines a vector which can be declared in generic code with the same representation-type as Vector.

  4. typedef Vector::SubvectorType SubvectorType;

    This defines a subvector of the given vector which can be constructed in generic code. It must support a constructor SubvectorType (v, start_index, size)

  5. typedef Vector::ConstSubvectorType ConstSubvectorType;

    Const version of Subvector

  6. typedef Vector::AlignedSubvectorType AlignedSubvectorType;

    This type may perform more quickly than Subvector but the choice of start_index is restricted to an integer-multiple of align. If align is one, then it is equivalent to Subvector.

    This is useful for recursive algorithms with some control over where a vector is to be split.

  7. typedef Vector::ConstAlignedSubvectorType ConstAlignedSubvectorType;

    Const version of AlignedSubvector

DefaultVectorTraits public public data members

  1. static const int align;

    The starting-index and possibly size of a AlignedSubvectorType must be an integral multiple of this value.
