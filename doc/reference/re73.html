LELA::DoubleWord — Class which given an integral data-type, constructs a data-type which is twice as wide.

Description

This class provides only bitwise binary operators &, |, and ^, shift operators >> and <<, and their assignment-equivalents. As in the C++-standard, shifting by greater than the size of this datatype (which is twice the size of its parameter) is undefined.
