Description

This defines the interface which matrices should satisfy. It is provided for documentation only.

MatrixInterface public types

  1. typedef const RawIndexedIterator ConstRawIndexedIterator;

    Since rows may also be indexed, this allows A[i][j] notation to be used.

    This may be omitted by an implementation if no Row type is available

MatrixInterface public construct/copy/destruct

  1. MatrixInterface();
    Default constructor.

    Builds empty 0x0 matrix. 

  2. MatrixInterface(size_t m, size_t n);
    Constructor with size.

    An implementation may omit this constructor if it makes no sense to provide it (e.g. with Submatrix or TransposeMatrix)

    Parameters:

    m

    row dimension

    n

    column dimension

  3. MatrixInterface(const MatrixInterface & M);
    Copy constructor. 
  4. MatrixInterface(VectorStream< Row > & vs);
    Constructor from a VectorStream.

    The VectorStream supplies either row- or column-vectors for the matrix, depending on the implementation. Its parameters also specify the size of the matrix.

    Parameters:

    vs

    VectorStream from which to get vectors 

  5. MatrixInterface& operator=(const MatrixInterface & M);
    Operator =.

MatrixInterface public member functions

    Get the number of columns in the matrix.

    Returns:

    Number of columns in matrix 

  3. void resize(size_t m, size_t n);
    Resize the matrix to the given dimensions.

    The state of the matrix's entries after a call to this method is undefined.

    An implementation may omit this interface if it makes no sense (e.g. for Submatrix or TransposeMatrix)

    Parameters:

    Number of rows

    n

    Number of columns
