Name

LELA::BLAS1::dot — Compute the dot-product of the two vectors.

Synopsis

// In header: </home/hovinen/build/lela/lela/blas/level1.h>


template<typename reference, typename Ring, typename Modules, 
         typename Vector1, typename Vector2> 
  reference & dot(Context< Ring, Modules > & ctx, reference & res, 
                  const Vector1 & x, const Vector2 & y);

Description

The optional parameters start_idx and end_idx allow the computation of the dot-product of just part of the vectors.

The vectors x and y may be of different types which different representations, but must be defined over the same ring.

Parameters:

ctx

Context object for calculation

res

Ring-element in which to place output

x

First vector

y

Second vector

Returns:

Reference to res
