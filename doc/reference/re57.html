Description

This module enables the naive algorithms used when nothing else is available.

Included in every module-structure should be an empty structure called Tag. Inside Tag should be a reference to the tag of the parent-module. With this mechanism the BLAS-methods are specialised to different modules.

All module-structures must have a constructor which takes a const Ring.
