Description

This class is a prototype of a vector meeting the dense interface in LELA. It can be used as a starting-point for those wishing to implement their own dense vector-types. It cannot be used on its own.

When feasible, suggested implementations of methods are provided.

DenseVectorInterface public types

  1. typedef Subvector SubvectorType;

    The subvector-type must support a construct of the form Subvector (v, start_index, end_index) where v is the parent-vector and start_index and end_index are indices of the beginning and end of the subvector.

    Normally it suffices to use Subvector for this type as given here.

  2. typedef Subvector< ConstIterator, ConstIterator > ConstSubvectorType;

    Normally it suffices to use Subvector for this type as given here.

  3. typedef Subvector AlignedSubvectorType;

    In nearly all cases, it should be identical to Subvector.

    Normally it suffices to use Subvector for this type as given here.

  4. typedef Subvector< ConstIterator, ConstIterator > ConstAlignedSubvectorType;

    Normally it suffices to use Subvector for this type as given here.

DenseVectorInterface public public data members

  1. static const int align;

    In nearly all cases -- and particularly when SubvectorType and AlignedSubvectorType are the same, it should be one.
